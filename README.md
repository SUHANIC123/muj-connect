# MUJ CONNECT

A comprehensive digital hub for Manipal University Jaipur students and faculty to connect, share, and stay updated with campus life.

### ğŸ¯ Key Features

- **Study Spots Directory**: Searchable and filterable list of campus study areas with noise level, WiFi availability, power outlets, and seating capacity information
